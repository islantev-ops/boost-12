## Развернуть у себя

```bash
git clone <адрес репозитория> "boost 12"
cd "boost 12/1-2-MVP/product-mvp"
npm install
npm run dev            # http://localhost:3000
```

Базы локально нет, и это ожидаемо: PostgreSQL живёт на VPS и слушает только
localhost. Интерфейс это переживает и честно пишет «База недоступна» — вёрстку и
логику смотреть можно.

Проверить любой сайт без базы и интерфейса:

```bash
npm run check-site https://example.ru
```

Полностью, с базой — см. [README приложения](1-2-MVP/product-mvp/README.md):
там установка PostgreSQL, `schema.sql`, `seed.sql` и деплой.

## Правило, которое важнее кода

Законы и суммы штрафов берутся **только из КонсультантПлюс**, каждая норма — со
ссылкой с якорем на конкретную часть статьи. Выдумывать нормы нельзя. Всё лежит
в одном месте: [src/lib/legal.ts](1-2-MVP/product-mvp/src/lib/legal.ts).

Состав проверок задан чек-листом на лендинге. Менять — сначала там, потом в коде.

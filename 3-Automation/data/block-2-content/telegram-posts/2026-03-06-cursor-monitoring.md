# Cursor теперь может мониторить вашу кодовую базу автономно и постоянно

**Дата:** 6 марта 2026
**Канал:** @data_secrets

---

Теперь там есть возможность настроить ИИ-агентов, которые автоматически выполняют задачи по расписанию или по событиям (push, Slack, PagerDuty и т.п.).

Каждый такой агент крутится в отдельной cloud-sandbox, может ходить в твой репозиторий, CI, Slack и внешние сервисы через MCP (Model Context Protocol).

При срабатывании триггера поднимается изолированная среда, подтягивается свежий код, и агент выполняет нужный сценарий. Настройки могут быть максимально гибкими.

Внутри уже есть типовые шаблоны: ежедневные дайджесты изменений, поиск уязвимостей и багов, обновление документации.

# Настройка на домашнем компьютере

Памятка: как поднять всё то же самое дома. Идите строго по шагам сверху вниз.

---

## Шаг 1. Поставить базовые программы

Их git не переносит — это программы уровня системы, ставятся один раз.

| Программа | Где взять |
|-----------|-----------|
| **VS Code** (редактор) | https://code.visualstudio.com — кнопка Download |
| **Claude Code** | расширение внутри VS Code (ставится в Шаге 3 автоматически) |
| **Git** | https://git-scm.com/download/win — обычно уже стоит вместе с VS Code |
| **GitHub CLI** (`gh`) | https://cli.github.com — кнопка Download, либо см. Шаг 4 |

> Скачивание может занять время — это нормально, делается один раз.

---

## Шаг 2. Войти в GitHub (чтобы скачать проект)

1. Открыть VS Code.
2. Слева внизу нажать на значок **человечка** (Accounts) → **Sign in with GitHub**.
3. Откроется браузер — подтвердить вход в аккаунт **islantev-ops**.

---

## Шаг 3. Включить синхронизацию настроек (расширения приедут сами)

Это заменяет «долгую установку» — все расширения и настройки подтянутся автоматически.

1. В VS Code слева внизу нажать **шестерёнку** ⚙️ → **Turn on Settings Sync…**
2. Выбрать вход через **GitHub**.
3. Нажать **Sign in & Turn on**.

После этого автоматически появятся, среди прочего:
- **Markdown Ognile** (`ognile.markdown-ognile`) — просмотр (превью) md-файлов. Это та самая фича.
- Русский языковой пакет.
- Claude Code.

> Если какого-то расширения вдруг нет — открыть «Расширения» (значок кубиков слева),
> вписать название (например `Ognile`) и нажать **Install**. Это 30 секунд.

---

## Шаг 4. Войти в GitHub CLI (для сохранения работы, по желанию)

Нужно, только если будете сохранять изменения из командной строки.

1. Открыть PowerShell (клавиша Windows → напечатать `powershell` → Enter).
2. Вставить и нажать Enter:
   ```
   gh auth login --web --git-protocol https
   ```
3. Появится код вида `1A2B-3C4D` → откроется браузер → вставить код → **Authorize**.
4. Дождаться в окне зелёной строки `Logged in as islantev-ops`.

---

## Шаг 5. Скачать проект

1. В VS Code нажать **Ctrl+Shift+P** → напечатать `Git: Clone` → Enter.
2. Вставить адрес репозитория:
   ```
   https://github.com/islantev-ops/boost-12.git
   ```
3. Выбрать папку, куда сохранить (например Рабочий стол).
4. Когда скачается — нажать **Open** (открыть папку).

Готово — проект дома, всё как на рабочем компьютере.

---

## Как сохранять новую работу (чтобы не потерялась)

После того как что-то поменяли в проекте:

**Простой способ — прямо в VS Code:**
1. Слева нажать значок **веток** (Source Control, третий сверху).
2. В поле сверху написать короткое описание что сделали.
3. Нажать **Commit** → затем **Sync Changes** (или **Push**).

**Или одной командой в PowerShell** (находясь в папке проекта):
```
git add -A; git commit -m "что сделал"; git push
```

Изменения улетят на GitHub и будут доступны на любом компьютере.
